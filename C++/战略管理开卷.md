# 企业战略管理开卷考试核心资料总结

## 一、绪论（Chapter 1）

### （一）战略管理基础概念

1. **战略管理的定位**战略管理是整合性管理理论，属于企业最高层次管理理论，是企业最高层管理人员最重要的活动和技能。

2. **管理者能力要求（罗伯特・卡茨）**

   * 技术能力（战术能力）：具体业务操作与执行能力
   * 人际关系能力（社会能力）：团队协作、沟通协调能力
   * 思维能力（战略能力）：全局视野、长期规划能力

3. **战略的起源与定义**

   * **起源**：源于希腊词 “strategos”（将军指挥军队的艺术）；克劳塞维茨《战争论》指出 “战略是为了达到战争的目的而对战斗的运用”；毛泽东《中国革命战争的战略问题》提出 “战略问题是研究战争全局的规律性的东西”。
   * **企业战略定义**：明确企业长期发展方向和活动范围，根据环境变化合理配置资源，通过满足利益相关者期望构筑竞争优势；是企业面对激烈变化的经营环境，为求得长期生存与发展，对目标、途径和手段的总体谋划。

4. **战略的 5P 观点（明茨伯格）**

   | 维度                | 核心内涵                       | 示例                                   |
   | ------------------- | ------------------------------ | -------------------------------------- |
   | Plan（计划）        | 事先对未来行动方案的说明与要求 | 企业 3 年数字化转型规划                |
   | Pattern（模式）     | 企业长期行为的一致性表现       | 华为持续投入研发的行为惯性             |
   | Position（定位）    | 企业在市场中的独特位置         | 小米 “高性价比智能硬件” 定位           |
   | Perspective（视角） | 对未来的预期，体现企业价值观   | 阿里 “让天下没有难做的生意” 的价值导向 |
   | Ploy（计谋）        | 竞争中采用的策略与手段         | 京东 618 大促的价格策略                |

### （二）战略的重要性与实践原则

1. **战略重要性相关理论**

   * 古语引用：“知止而后定，定而后能静，静而后能安，安而后能虑，虑而后能得”“上兵伐谋，其次伐交，其次伐兵，其下攻城”，强调战略优先性。
   * **5% 原则**：哈佛商学院约翰・威尔斯提出企业 CEO 每天应花 5% 时间思考战略；IBM 高管团队每月花 1 天回顾战略；未遵循此原则的企业常因危机才反思战略，错失变革机会。

2. **企业战略实践陷阱**

   | 陷阱类型             | 核心问题                                     | 示例                                      |
   | -------------------- | -------------------------------------------- | ----------------------------------------- |
   | 指挥中枢陷阱（高层） | 缺乏统一战略逻辑、高管同质思维、一线无决策权 | 某国企因高层决策固化，错失数字化转型机遇  |
   | 适应系统陷阱（能力） | 技术 / 模式创新滞后、动态能力不足            | 柯达因胶卷技术刚性，无法应对数码时代变革  |
   | 运行系统陷阱（治理） | 委托代理限制激励、集团总部空心化             | 某家族企业因治理机制僵化，核心人才流失    |
   | 执行主体陷阱（人力） | 唯金钱激励、考核体系低效、人才视为成本       | 某民企因薪酬机制单一，员工创新动力不足    |
   | 内在意识陷阱（文化） | 科层制削弱市场响应、创业精神弱化             | 某大企业因 “大企业病”，新产品迭代速度落后 |

### （三）战略管理理论演变与学派

1. **理论演变阶段**

   * 早期战略思想阶段（20 世纪初 - 50 年代）：侧重经验总结，无系统理论
   * 古典战略理论阶段（20 世纪 60 年代 - 80 年代初）：安索夫《Corporate Strategy》（1965）提出战略管理框架，标志企业战略管理诞生（3C 时期：Change、Complexity、Conflict）
   * 竞争战略理论阶段（20 世纪 80 年代以来）：波特竞争战略（定位、取舍、配称）
   * 动态复杂战略理论阶段（20 世纪 90 年代以来）：动态能力理论、生态系统理论

2. **战略十大学派（明茨伯格）**

   | 阶段                   | 学派           | 核心观点                                                     | 代表方法 / 人物                |
   | ---------------------- | -------------- | ------------------------------------------------------------ | ------------------------------ |
   | 第一阶段（理性主义）   | 设计学派       | 战略是扬长避短、趋利避害的组合                               | SWOT 分析                      |
   |                        | 计划学派       | 战略制定是正规计划过程                                       | 定量分析模型                   |
   |                        | 定位学派       | 战略目的是获取高于行业平均利润，取决于行业盈利能力与企业位置 | 迈克尔・波特，五力模型、价值链 |
   | 第二阶段（非理性主义） | 自由企业家学派 | 战略是企业家对未来的愿景（Visionary）                        | 强调企业家直觉                 |
   |                        | 认知学派       | 战略制定含理性与非理性思维                                   | 关注认知盲区、思维局限         |
   |                        | 学习学派       | 战略是个人 / 集体学习的结果，适应环境变化                    | 注重组织学习过程               |
   |                        | 权力学派       | 战略是利益团体权力博弈、谈判的结果                           | 关注正式 / 非正式权力影响      |
   |                        | 文化学派       | 战略受企业文化、社会价值观影响                               | 强调组织文化对战略的约束       |
   |                        | 环境学派       | 环境是战略核心，企业需适应环境                               | 侧重环境对战略的决定作用       |
   | 第三阶段（整合）       | 整合学派       | 战略制定如 “盲人摸象”，需整合各学派观点                      | 明茨伯格，强调全面视角         |

## 二、企业外部环境分析（Chapter 2）

### （一）宏观环境分析（PESTEL 模型）

1. **六大维度及核心要素**

   | 维度                  | 核心分析要素                                         | 案例 / 关注重点                                              |
   | --------------------- | ---------------------------------------------------- | ------------------------------------------------------------ |
   | 政治（Political）     | 政治制度、政策法规、政府稳定性、腐败程度             | 全球清廉指数（丹麦、芬兰常年领先；中国 2021 年得分 45，排名 63）；电信、航空行业准入管制 |
   | 经济（Economic）      | GDP 增速、人均收入、通胀率、消费率、中等收入群体规模 | 中国 “双循环” 格局；2025 年人均收入预测；消费率与投资率变化  |
   | 社会（Social）        | 人口结构（年龄、城镇化）、教育水平、消费偏好         | 16-59 岁劳动年龄人口下降；常住人口城镇化率提升；消费者对绿色产品需求增长 |
   | 技术（Technological） | 新技术突破（AI、5G、区块链）、研发投入、技术迭代速度 | 新基建（5G 基站、特高压、大数据中心）；数码技术淘汰胶卷行业  |
   | 自然（Environmental） | 能源消费、环保政策、可再生能源、气候变化             | 碳中和目标；绿色经济；环境友好型产品生产                     |
   | 法律（Legal）         | 劳动法、税法、知识产权保护、反垄断法                 | 数据安全法；平台经济反垄断监管；知识产权侵权风险             |

2. **环境分析步骤**扫描（发现环境变化早期信号）→ 监测（持续观察变化意义）→ 预测（形成结果预期）→ 评估（判断对战略的重要性）

### （二）行业环境分析（核心工具）

1. **行业基本竞争结构（波特五力模型）**

   | 竞争力量       | 影响因素                                                     | 应对策略                               |
   | -------------- | ------------------------------------------------------------ | -------------------------------------- |
   | 现有企业间竞争 | 竞争者数量、行业增长速度、产品同质化、退出障碍、固定成本     | 差异化、成本领先、提高退出壁垒         |
   | 潜在进入者威胁 | 规模经济、产品差异（品牌忠诚）、资金需求、转换成本、政策限制 | 构建规模壁垒、强化品牌、控制渠道       |
   | 供应商议价能力 | 供应商集中度、产品标准化、转换成本、前向一体化能力           | 多元化供应商、建立长期合作、后向一体化 |
   | 购买者议价能力 | 购买者集中度、产品标准化、转换成本、后向一体化能力           | 产品差异化、提高转换成本、提供增值服务 |
   | 替代品威胁     | 替代品盈利能力、用户转换成本、替代品企业战略                 | 技术创新、提升产品性价比、构建用户粘性 |

2. **行业生命周期分析**

   | 阶段   | 特征                                   | 战略重点                           |
   | ------ | -------------------------------------- | ---------------------------------- |
   | 导入期 | 市场小、增长慢、成本高、竞争少         | 技术研发、培育市场、建立品牌       |
   | 成长期 | 市场快速增长、竞争加剧、利润上升       | 扩大产能、抢占市场份额、优化渠道   |
   | 成熟期 | 市场饱和、增长缓慢、竞争激烈、利润稳定 | 成本控制、产品创新、细分市场       |
   | 衰退期 | 市场萎缩、需求下降、竞争减少           | 业务收缩、退出或转型、收割短期利润 |

3. **行业关键成功因素（KSF）**行业内企业成功经营必须具备的关键能力 / 资源，如：

   * 汽车行业：技术研发、供应链管理
   * 快消行业：品牌营销、渠道覆盖

### （三）竞争对手分析

1. **分析框架（“知彼知己”）**
   * 竞争对手目标：管理层目标、财务目标、市场目标
   * 竞争对手现行战略：成本领先、差异化、聚焦
   * 竞争对手能力：强项（核心能力）、弱项（资源短板）
   * 竞争对手反应：对现状满意度、报复策略、脆弱点
2. **竞争态势矩阵（CPM）**
   * 步骤：识别关键战略因素→赋予权重（0-1，按重要性）→评分（1-4，企业相对表现）→计算加权分→总加权分对比（得分越高竞争力越强）
   * 实例：某行业关键因素含市场份额（权重 0.2）、价格竞争力（0.2）、财务状况（0.4），总加权分 2.8 高于 2.1/2.2，说明竞争力更强

### （四）战略群体与外部因素评价

1. **战略群体分析**
   * 定义：行业中实施相同 / 相似战略的一组企业
   * 绘制步骤：辨析差别化因素（价格 / 质量、地理范围）→ 双变量图列厂商→归为同一战略群→画圆（半径 = 销售收入份额）
   * 意义：识别直接竞争对手（同群体）、分析群体间竞争强度、判断群体转移壁垒
2. **外部因素评价矩阵（EFE）**
   * 步骤：列出关键外部因素（10-20 个，机会 + 威胁）→ 赋权重（0-1）→ 评分（1-4，企业应对有效性）→ 计算加权分→ 总加权分（1.0-4.0，2.5 为平均，高于则优于平均）
   * 实例：某企业总加权分 2.64，说明利用机会、回避威胁能力略高于行业平均

## 三、企业内部环境分析（Chapter 3）

### （一）企业资源与能力分析

1. **企业资源分类**

   | 资源类型 | 具体内容                                                     | 特征对比（物质资源 vs 知识资源）                             |
   | -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
   | 有形资源 | 物质资产（设备、厂房）、金融资产（资金、股票）               | 物质资源：稀缺、有限、排他、复制成本高；知识资源：丰富、无限、共享、复制成本低、效用递增 |
   | 无形资源 | 技术资源（专利、技术）、关系资源（客户、供应商）、声誉资源（品牌、口碑） | 价值增值性强，难以模仿，是竞争优势核心来源                   |
   | 人力资源 | 员工知识、技能、创新能力、协作能力                           | “知道为什么做” 比 “知道怎么做” 更关键，是知识资源的载体      |

2. **企业能力分类**

   | 能力类型       | 具体能力                                                     | 实例                                               |
   | -------------- | ------------------------------------------------------------ | -------------------------------------------------- |
   | 职能领域能力   | 营销能力（渠道管理、品牌建设）、财务能力（现金流、资本运作）、研发能力（产品革新） | 华为研发能力（5G 技术）、茅台品牌营销能力          |
   | 跨职能综合能力 | 学习能力（组织学习、自适应）、创新能力（产品 / 模式创新）、整合能力（供应链整合） | 小米生态链整合能力、阿里数字化创新能力             |
   | 动态能力       | 适应环境变化、资源重构、快速响应竞争                         | 腾讯从社交到生态的转型能力、字节跳动全球化调整能力 |

3. **资源基础观（RBV）核心观点**企业资源差异是获利能力不同的原因；可持续竞争优势需资源满足**VRIO 标准**：

   * Valuable（有价值）：帮助把握机会、化解威胁
   * Rare（稀缺）：很少被其他企业拥有
   * Inimitable（难以模仿）：因历史积累、社会复杂性、模糊性难以模仿
   * Organized（可组织利用）：通过组织架构、激励机制转化为竞争优势

### （二）企业核心能力

1. **定义（普拉哈拉德 & 哈默尔）**企业内一组技术和技能的集合（含硬技术 + 软技术），长期积累形成，可实现高于对手的价值，难以复制，是长期利润源泉（如耐克的品牌运营 + 供应链管理、华为的研发 + 客户服务）。
2. **核心能力判断标准（VRIO 延伸）**
   * 有价值（Value）：帮助把握机会、化解威胁
   * 稀缺性（Rarity）：很少被其他企业拥有
   * 难以模仿性（Inimitability）：因历史积累、社会复杂性（人际关系、信任）、模糊性（来源不明）难以模仿
   * 不可替代性（Non-substitutability）：无战略等价资源替代
3. **核心能力与产品的关系**核心能力（树根）→ 核心产品（树干）→ 最终产品（枝叶 / 花果），如佳能核心能力（光学技术）→ 核心产品（镜头）→ 最终产品（相机、打印机）。
4. **核心能力风险**核心刚性（过去核心能力成为未来变革障碍，如柯达胶卷技术拖累数码转型）；需动态更新核心能力，避免老化。

### （三）价值链分析（波特）

1. **价值链构成**

   | 活动类型 | 具体活动                                                     | 实例（小米）                                                 |
   | -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
   | 基本活动 | 内部后勤（原材料采购、库存管理）、生产作业（制造、组装）、外部后勤（仓储、运输）、市场销售（营销、渠道）、售后服务（维修、客服） | 小米供应链管理（内部后勤）、线上线下销售（市场销售）、小米之家售后 |
   | 支持活动 | 企业基础结构（战略规划、财务）、人力资源管理（招聘、培训）、技术开发（产品设计、工艺）、采购（原材料、设备采购） | 小米战略规划（生态链布局）、技术开发（MIUI 系统）、供应商采购 |

2. **价值链分析应用**

   * 内部价值链：识别价值增值关键环节（如华为研发、苹果设计），优化或外包非关键环节
   * 外部价值链：整合供应商、渠道、顾客价值链（如美的 “全渠道一盘货”，共享库存提高效率）
   * 价值链 vs 生态链：传统价值链是单向线性（研发→生产→销售），生态链是多主体共生（小米生态链：手机→IoT 设备→内容服务）

### （四）内部评价工具

1. **内部因素评价矩阵（IFE）**
   * 步骤：列出关键内部因素（优势 + 劣势，10-20 个）→ 赋权重（0-1）→ 评分（1-4，1 = 重要劣势，4 = 重要优势）→ 计算加权分→ 总加权分（1.0-4.0，2.5 为平均，高于则内部状况好）
   * 实例：某企业总加权分 2.80，说明内部战略地位高于行业平均
2. **SWOT 分析**
   * 矩阵构建：优势（S）、劣势（W）、机会（O）、威胁（T）
   * 战略匹配：
     * SO 战略：利用内部优势 + 把握外部机会（如华为利用研发优势 + 5G 机会）
     * WO 战略：利用外部机会 + 弥补内部劣势（如传统企业利用数字化机会弥补转型能力不足）
     * ST 战略：利用内部优势 + 规避外部威胁（如小米利用生态链优势规避价格战威胁）
     * WT 战略：减少内部劣势 + 规避外部威胁（如濒临破产企业收缩业务 + 退出高风险市场）

## 四、企业使命、愿景与战略目标（Chapter 4）

### （一）使命（Mission）

1. **定义**企业根本性性质与存在目的，回答 “我们的业务是什么？”，为战略制定提供依据（如阿里巴巴 “让天下没有难做的生意”、迪斯尼 “让人们过得快活”）。
2. **使命陈述核心内容**
   * 经营目的：长期价值追求（如华为 “丰富人们的沟通和生活”）
   * 经营范围：行业 / 市场边界（如小米 “智能硬件 + IoT + 互联网服务”）
   * 经营方式：独特竞争方式（如宜家 “自助购物 + 平板包装”）
   * 竞争优势领域：资源配置重点（如华为研发投入）
   * 利益相关者承诺：顾客、员工、股东、社会（如 “四满意” 原则）
3. **使命的作用**明确方向、协调分歧、吸引资源、体现独特性、导向顾客与社会责任。

### （二）愿景（Vision）

1. **定义**企业对未来的憧憬与渴望，回答 “我们想成为什么？”（如福特早期 “让每个家庭都有一辆汽车”、华为 “构建万物互联的智能世界”）。
2. **愿景确立原则**
   * 宏伟：超越常态，激动人心（如腾讯 “用户为本，科技向善”）
   * 振奋：语言热烈，感染员工（如迪斯尼 “成为世界上最快乐的地方”）
   * 清晰：形象生动，直觉认知（如福特 “使汽车大众化”）
   * 可实现：激发信念，非空想（如北交大经管学院 “成为引领中国产业发展的商学院”）

### （三）战略目标

1. **定义**企业在一定时期内，依据使命与愿景，结合内外部条件的预期成果（非手段，如 “2025 年营收破千亿” 是目标，“采用大数据” 是手段）。

2. **目标体系分类**

   | 分类维度 | 目标类型                                                     | 实例                                                         |
   | -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
   | 主体层次 | 企业总目标、经营单位目标、部门目标                           | 企业总目标 “2025 年营收破千亿”→子公司目标 “华东区域营收破 200 亿” |
   | 目标性质 | 市场目标（市场份额、客户数量）、创新目标（新产品数量、专利）、盈利目标（利润率、ROE）、社会目标（环保投入、就业岗位） | 小米市场目标 “全球手机市场份额 Top3”、盈利目标 “毛利率维持在 15% 以上” |

3. **目标制定原则（SMART）**

   * Specific（具体）：明确内容（如 “提升市场份额”→“2024 年华东区域市场份额从 15% 提升至 20%”）
   * Measurable（可衡量）：量化指标（如 “提高客户满意度”→“客户满意度从 85 分提升至 90 分”）
   * Attainable（可实现）：结合资源能力
   * Relevant（相关）：与使命、愿景一致
   * Time-based（时限）：明确完成时间（如 “2025 年底前完成数字化转型”）

## 五、竞争战略（Chapter 5）

### （一）基本竞争战略（波特三大战略）

1. **成本领先战略**
   * **定义**：通过规模经济、技术优化、成本控制，在行业内保持整体成本领先，以最低价格定价（薄利多销）。
   * **实施途径**：
     * 优化价值链：简化产品设计、削减附加服务、直接营销（如春秋航空 “单一机型、无免费餐食、自助登机”）
     * 管理成本驱动因素：规模经济（大批量生产）、学习效应（积累经验）、控制采购成本（集中采购）
   * **适用条件**：市场需求价格弹性高、产品标准化、行业产能过剩、企业有大规模生产能力。
   * **风险**：技术变革淘汰设备、用户偏好转向非价格因素、竞争对手模仿成本控制方法。
2. **差异化战略**
   * **定义**：使产品 / 服务有别于对手，在全行业树立独特性，提升感知价值（Perceived Value），获得高价格。
   * **实施途径**：
     * 产品差异化：性能、设计、质量（如谭木匠 “手工木梳 + 文化内涵”）
     * 营销差异化：品牌、渠道、服务（如苹果 “高端品牌 + 体验店服务”）
     * 人员 / 形象差异化：员工专业度、企业形象（如星巴克 “第三空间” 形象）
   * **适用条件**：客户需求多样化、产品差异化易被感知、企业有研发 / 营销能力、行业竞争激烈。
   * **风险**：顾客支付溢价极限、模仿缩小差异、过度差异化（价格过高 / 超出需求）。
3. **目标集中战略（聚焦战略）**
   * **定义**：聚焦特定顾客群、产品细分市场或区域市场，分 “成本集中”（如 7 天连锁酒店 “经济型酒店 + 商务人群”）和 “差异化集中”（如好孩子童车 “儿童市场 + 安全设计”）。
   * **适用条件**：目标市场有一定规模 / 增长潜力、非主要竞争厂家重点、企业有服务目标市场的资源。
   * **风险**：规模不经济、目标市场需求变化、竞争对手进入细分市场。

### （二）整合战略（最优成本战略）

1. **定义**结合成本领先与差异化，以较低成本提供较高质量产品，注重性价比（如小米 “高配置 + 中价格”、丰田雷克萨斯 “豪华配置 + 低于 BBA 价格”）。
2. **实施途径**：
   * 技术支撑：柔性制造（多品种小批量 + 规模经济）、标准化零部件（降低成本 + 保证质量）
   * 管理优化：准时制生产（JIT，减少库存）、数字化管理（大数据优化成本与差异）。
3. **风险**：陷入 “夹在中间”，既无成本优势也无差异化优势，受两端企业夹击。

### （三）蓝海战略

1. **核心思想**摆脱 “红海”（已知市场血腥竞争），开创 “蓝海”（新市场空间），通过**价值创新（Value Innovation）** 为买方与企业创造价值飞跃（非打败对手）。
2. **价值曲线四步动作框架**
   * 剔除：删除行业公认但无价值的要素（如太阳马戏团剔除 “动物表演”）
   * 减少：降低行业过度投入的要素（如太阳马戏团减少 “明星演员”）
   * 增加：提高行业投入不足的要素（如太阳马戏团增加 “艺术氛围”）
   * 创造：新增行业未提供的要素（如太阳马戏团创造 “故事剧情”）
3. **实例**：太阳马戏团：整合马戏与戏剧，创造新娱乐市场；小红书：从 “海淘攻略” 到 “社区电商”，开创内容 + 交易新场景。

### （四）动态竞争

1. **定义**
   * 狭义：行业内企业竞争行动→对手反应→再影响原企业（如华为 vs 小米价格战）
   * 广义：超竞争环境下，竞争优势暂时化，创新与速度替代规模成为优势来源。
2. **特点**：先动优势暂时、竞争互动高速、战略有效性依赖预测对手反应。
3. **博弈论应用**：
   * 囚徒困境：单次博弈易 “降价竞争”，多次博弈易 “合作共赢”（如双寡头价格竞争）
   * 竞合理论：共存共生、共荣共赢，强调合作（如平台企业与合作伙伴）。

## 六、公司总体战略（Chapter 6）

### （一）战略态势选择

| 战略类型                     | 特征                             | 适用场景                   | 实例                                       |
| ---------------------------- | -------------------------------- | -------------------------- | ------------------------------------------ |
| 增长型战略（52.8% 企业采用） | 扩大规模、拓展业务、市场份额提升 | 行业增长期、企业有资源能力 | 华为全球化、阿里生态扩张                   |
| 稳定型战略（9.2%）           | 维持现状、小幅调整、规避风险     | 行业成熟期、企业资源有限   | 老干妈（聚焦辣酱，不多元化）               |
| 紧缩型战略（7.2%）           | 业务收缩、资产剥离、退出市场     | 行业衰退期、企业亏损严重   | 诺基亚退出手机业务、通用电气剥离非核心业务 |
| 混合型战略（28.7%）          | 部分业务增长、部分稳定 / 收缩    | 大型多元化企业             | 三星（电子增长 + 化工稳定）                |

### （二）密集型成长战略（专业化战略）

1. **定义**：集中资源于核心业务，通过专注某一点带动成长（如九阳 “从豆浆机到小家电领导者”）。

2. **实施路径（安索夫矩阵）**

   | 市场 \ 产品 | 现有产品                                             | 新产品                                             |
   | ----------- | ---------------------------------------------------- | -------------------------------------------------- |
   | 现有市场    | 市场渗透（提高现有产品市场份额，如可口可乐增加广告） | 产品开发（现有市场推出新产品，如苹果 iPhone 迭代） |
   | 新市场      | 市场开发（新市场推广现有产品，如华为进入欧洲市场）   | 多元化（新市场 + 新产品，如阿里从电商到云计算）    |

3. **优势**：定位清晰、资源集中、易形成核心能力；**劣势**：风险集中（如行业衰退则企业受损）。

### （三）一体化战略

1. **类型与实例**

   | 类型              | 定义                              | 实例                                                | 优势                             | 风险                           |
   | ----------------- | --------------------------------- | --------------------------------------------------- | -------------------------------- | ------------------------------ |
   | 纵向一体化 - 后向 | 向供应商领域拓展（控制原料）      | 中粮 “从田间到餐桌”（自建农场）、杰尼亚（自办牧场） | 降低采购成本、保证供应           | 产能失衡、管理复杂、退出壁垒高 |
   | 纵向一体化 - 前向 | 向买方领域拓展（控制渠道 / 终端） | 格力 “自建专卖店”、耐克 “直营门店”                  | 控制分销、提升品牌、掌握客户     | 渠道建设成本高、与经销商冲突   |
   | 横向一体化        | 并购 / 联盟同类企业（扩大规模）   | 滴滴并购快的、美团并购摩拜                          | 扩大市场份额、规模经济、减少竞争 | 反垄断监管、整合难度大         |

2. **实施条件**：行业竞争激烈、上下游利润高、企业有整合能力、交易成本高（需内部化）。

### （四）多元化战略

1. **定义与分类**

   | 分类         | 判断标准                          | 实例                                                         | 优势                         | 风险                               |
   | ------------ | --------------------------------- | ------------------------------------------------------------ | ---------------------------- | ---------------------------------- |
   | 相关多元化   | 新业务与现有业务有技术 / 市场关联 | 小米 “手机→IoT 设备”（技术关联）、海尔 “冰箱→洗衣机”（市场关联） | 范围经济、资源共享、风险分散 | 管理跨度大、协同效应不足           |
   | 非相关多元化 | 新业务与现有业务无关联            | 娃哈哈 “饮料→童装→白酒”、通用电气 “家电→航空发动机”          | 分散风险、捕捉新机会         | 学习成本高、核心能力弱化、财务风险 |

2. **多元化程度判断**

   | 某类产品销售额占比 | 企业类型   | 多元化程度 |
   | ------------------ | ---------- | ---------- |
   | >95%               | 单一事业型 | 专业化     |
   | 95%-70%            | 主导事业型 | 低度多元化 |
   | <70%（产品相关）   | 相关多元化 | 中度多元化 |
   | <70%（产品无关）   | 无关多元化 | 高度多元化 |

3. **多元化五问（成功关键）**基础稳（现有业务优势）→ 进得去（有进入优势）→ 站得住（超越对手）→ 无冲突（不破坏现有优势）→ 有发展（为未来奠基）。

### （五）平台战略与数字化战略

1. **平台战略**
   * **定义**：连接多方用户（供给方 + 需求方），构建生态系统，实现网络效应（如淘宝 “卖家 + 买家”、微信 “用户 + 服务商”）。
   * **构建要素**：定位多边市场、激发网络效应（同边：用户越多越吸引新用户；跨边：供给方越多吸引更多需求方）、设定付费 / 补贴方（如滴滴补贴乘客吸引司机）、筑起用户过滤机制（如淘宝店铺评级）。
   * 与传统模式对比：传统是线性价值链（作者→出版社→读者），平台是多边互动（作者群→网络出版平台→读者群），成本低、效率高。
2. **数字化战略**
   * **定义**：利用大数据、AI、云计算等，重构现实世界为数字世界，实现生产要素 “新组合”（熊彼特创新理论）。
   * **数字化转型方向**：数字营销（精准定位）、智能生产（柔性制造）、数字商业模式（数据变现）、智能管控（数据分析决策）。
   * **实例**：海尔 COSMOPlat（工业互联网平台）、美的 “T+3” 模式（以销定产，数字化供应链）。

## 七、并购战略（Chapter 7）

### （一）并购定义与类型

1. **定义**
   * 兼并：吸收合并（A 吸收 B，B 解散）、新设合并（A+B 成立 C）
   * 收购：现金 / 有价证券购买另一家公司股权，纳入战略组合（如联想收购 IBM PC 业务）。
2. **全球并购趋势**：2022 年全球并购盘整，中国交易总额 4858 亿美元，54 宗超 10 亿美元并购。

### （二）并购动因（为何选并购而非自建）

1. 突破行业进入障碍（如电信、航空需并购进入）
2. 加快市场进入速度（如联想通过并购 IBM PC 快速国际化）
3. 增强市场份额与影响力（如中国电信收购联通 C 网）
4. 获取技术 / 资源（如吉利收购沃尔沃获取豪华车技术）
5. 克服多元化难处（快速进入新行业，减少学习成本）

### （三）并购失败原因（成功率仅 20%）

1. 战略目标不明确：盲目并购，与企业战略脱节
2. 评估不充分：尽职调查（财务、文化、整合难度）不到位
3. 债务负担过重：交易带来超出能力的债务
4. 协同效应不足：无法实现规模经济 / 资源共享（如车企并购后研发协同失败）
5. 过度多元化：管理跨度大，短期利润导向削弱长期竞争力
6. 整合困难：战略、文化、业务、人力资源整合受阻（如联想收购摩托罗拉后文化冲突）

### （四）成功并购要素

1. 明确战略目标：并购与长期战略一致
2. 充分尽职调查：评估财务、文化、整合效益与难度
3. 友好并购：减少对抗，降低整合成本（如吉利收购沃尔沃保持其品牌独立性）
4. 资金充分：避免债务风险
5. 整合策略：
   * 文化整合：尊重被并购方文化，逐步融合
   * 业务整合：共享研发、采购等关键环节
   * 人力资源整合：保留核心人才，明确员工安排
6. 关注长期创新：不将并购作为创新替代品

### （五）案例：吉利收购沃尔沃

1. **背景**：吉利需提升技术 / 品牌，沃尔沃需资金 / 中国市场。
2. **准备**：尽职调查、组建专业团队、争取政府支持。
3. **整合举措**：保持沃尔沃独立性、共享技术平台（CMA 架构）、拓展中国市场。
4. **启示**：明确战略目标、尊重被并购方、重视技术协同。

## 八、战略联盟（Chapter 8）

### （一）战略联盟定义与类型

| 类型       | 定义                          | 实例                                       | 适用场景                         |
| ---------- | ----------------------------- | ------------------------------------------ | -------------------------------- |
| 股权式联盟 | 以股权为纽带，合资 / 相互持股 | 一汽 - 大众（合资）、腾讯持股京东          | 长期合作、转移隐性知识（如技术） |
| 契约式联盟 | 通过协议共享资源，无股权      | 苹果与富士康（代工）、耐克与供应商（外包） | 短期合作、避免核心知识泄露       |

### （二）联盟动因（按市场类型）

1. **慢周期市场（垄断 / 管制行业，如电力、铁路）**：进入管制市场、建立市场标准、保持稳定性。
2. **快周期市场（竞争充分，如 IT）**：加快新产品开发 / 上市、分担研发投入、减少市场不确定性。
3. **标准周期市场（如汽车、家电）**：获得互补资源、规模经济、克服贸易壁垒、应对竞争。

### （三）联盟风险（70% 失败，平均寿命 7 年）

1. 信任不足：合作方机会主义行为（如泄露技术）
2. 利益冲突：目标不一致（如一方追短期利润，一方追长期技术）
3. 能力失衡：一方投入不足，依赖另一方
4. 整合困难：文化、管理模式差异
5. 伙伴变对手：横向联盟中合作方转为竞争者（如华为与中兴曾合作后竞争）

### （四）成功联盟策略（PARTS 模型）

1. 评估伙伴（Player）：选择有诚意、资源互补的伙伴
2. 明确增值（Added values）：确定双方贡献与收益
3. 制定规则（Rules）：明确目标、责任、权利，签订详细合约
4. 灵活战术（Tactics）：适应市场变化，调整合作方式
5. 界定范围（Scope）：明确合作时空范围，避免过度依赖
6. 其他：建立信任关系、保持学习能力、把握联盟升级机会

### （五）案例：宁德时代战略联盟

1. **动因**：保障电池需求（与车企联盟）、获取技术（与高校 / 科研机构联盟）。
2. **联盟网络**：与宝马、特斯拉等车企合资建厂；与中科院合作研发。
3. **风险**：车企自建电池厂（如比亚迪）、技术路线变化（如固态电池）。
4. **应对**：持续研发、拓展储能领域、多元化客户。

## 九、国际化经营战略（Chapter 9）

### （一）国际化动因

1. 扩大市场：大市场孵化大企业（如华为进入欧洲、印度市场）
2. 获取稀缺资源：如力拓矿业全球获取矿产
3. 实现区位经济：全球配置价值链（如宜家全球采购 + 集中设计）
4. 利用规模经济和经验效应：如苹果全球生产降低成本
5. 复制核心能力：如麦当劳将标准化模式复制到全球
6. 延长产品生命周期：如欧美成熟产品引入新兴市场

### （二）公司级国际化战略类型（按成本压力与地区调适压力）

| 战略类型             | 成本压力 | 地区调适压力 | 核心特点                           | 实例                            |
| -------------------- | -------- | ------------ | ---------------------------------- | ------------------------------- |
| 本土化战略（多国化） | 低       | 高           | 授权当地业务单元，提供本土化产品   | 肯德基中国（推出油条、粥）      |
| 全球标准化战略       | 高       | 低           | 全球统一产品 / 流程，追求规模经济  | 微软 Windows 系统（全球统一）   |
| 跨国战略             | 高       | 高           | 兼顾全球效率与本土反应，“弹性协调” | 卡特彼勒（集中生产 + 本土改造） |
| 国际战略             | 低       | 低           | 母国创新，海外销售标准化产品       | 早期宝洁（美国研发，全球销售）  |

### （三）国际市场进入方式

| 进入方式                      | 优势                               | 劣势                       | 适用场景                                  |
| ----------------------------- | ---------------------------------- | -------------------------- | ----------------------------------------- |
| 出口                          | 低风险、高灵活性、规模经济         | 高运输费、贸易壁垒         | 早期进入、产品标准化（如家电）            |
| 技术授权 / 特许经营           | 低开发成本、快速进入               | 无控制权、不能实现规模经济 | 技术 / 模式可复制（如可口可乐特许灌装）   |
| 合资经营                      | 利用当地知识、分担风险、政治可接受 | 无控制权、文化冲突         | 需适应本土政策 / 文化（如汽车合资）       |
| 全资子公司（收购 / 绿地投资） | 有控制权、保护技术、全球协同       | 高成本 / 风险、需整合能力  | 需控制核心技术 / 品牌（如华为海外子公司） |
| 战略联盟                      | 资源互补、降低风险、克服壁垒       | 信任风险、利益冲突         | 需快速获取资源（如车企与当地经销商联盟）  |

### （四）国际化风险与挑战

1. **政治风险**：政府不稳定、战争、国有化、政策变革（如俄罗斯政局不稳影响外资）
2. **经济风险**：汇率波动、通胀、金融管制、知识产权保护弱（如东南亚知识产权侵权）
3. **管理复杂性**：地域分散、协调成本高、文化差异、东道国政府限制（如谷歌退出中国因文化 / 政策）

### （五）波特钻石理论（影响国际竞争优势的因素）

| 要素                 | 核心内容                                   | 实例（意大利制鞋业）                 |
| -------------------- | ------------------------------------------ | ------------------------------------ |
| 生产要素             | 高级要素（熟练工人、数字技术）、专业化要素 | 制鞋熟练工人、皮革加工技术           |
| 需求状况             | 本国市场需求规模 / 属性，提炼核心能力      | 意大利消费者对鞋品质要求高，推动创新 |
| 相关与支持产业       | 上下游产业支持                             | 皮革加工、鞋机制造、设计服务发达     |
| 企业战略、结构与竞争 | 企业战略制定、组织架构、国内竞争强度       | 意大利制鞋企业竞争激烈，推动差异化   |

## 十、战略的评价与选择（Chapter 10）

### （一）战略评价方法

1. **波士顿矩阵（BCG Matrix）**

   * **维度**：相对市场占有率（横轴，0.5 为界）、市场增长率（纵轴，10% 为界）

   * **业务类型与战略**

     | 象限 | 业务类型 | 特征                           | 战略选择                  |
     | ---- | -------- | ------------------------------ | ------------------------- |
     | I    | 问号类   | 高增长、低份额，需大量投资     | 发展（提升份额）或放弃    |
     | II   | 明星类   | 高增长、高份额，需投资击退对手 | 发展（保持领先）          |
     | III  | 现金牛类 | 低增长、高份额，现金流来源     | 维持 / 收获（稳定现金流） |
     | IV   | 瘦狗类   | 低增长、低份额，盈利差         | 放弃（收缩业务）          |

   * **优点**：简单明了，指导资源配置；**缺点**：仅两指标，静态分析。

2. **行业吸引力 - 企业实力矩阵（IE Matrix）**

   * **维度**：行业吸引力（市场大小、增长率等，加权评分分大 / 中 / 小）、企业实力（市场份额、研发等，加权评分分强 / 中 / 弱）
   * **战略匹配**：
     * 行业吸引力大 + 企业实力强（I 象限）：增长型战略
     * 行业吸引力中 + 企业实力中（V 象限）：稳定型战略
     * 行业吸引力小 + 企业实力弱（IX 象限）：收缩型战略
   * **优点**：多因素评价，更全面；**缺点**：权重确定主观。

3. **定量战略规划矩阵（QSPM Matrix）**

   * **步骤**：列出关键内外部因素→赋权重→对各战略评分（AS，0-4，吸引力）→计算加权分（TAS）→总 TAS 最高者为优选战略
   * **优点**：定量分析，减少主观；**缺点**：依赖因素选择与权重设定。

### （二）影响战略选择的行为因素

1. 过去战略的影响：路径依赖，难大幅调整
2. 利益集团压力：股东（短期利润）、员工（岗位稳定）、政府（就业 / 环保）
3. 企业领导人的价值观和对待风险的态度：冒险型选增长战略，保守型选稳定战略
4. 企业内部的人事和权力因素：高管团队权力斗争影响战略
5. 时间因素：战略调整需时间，紧急情况难从容选择
6. 竞争对手的反应：需预测对手应对，避免恶性竞争

## 十一、企业战略实施（Chapter 11）

### （一）战略实施基本原则

1. 适度合理性原则：战略需调整以适应实际情况，非僵化执行
2. 统一领导、统一指挥原则：避免多头领导，确保指令一致
3. 权变原则：根据环境变化动态调整实施策略（“流动的水”）

### （二）战略实施主要任务

1. 组织保障：建立与战略匹配的组织结构（如多元化企业用事业部制）
2. 文化建设：创造与战略吻合的企业文化（如创新战略需鼓励冒险的文化）
3. 资源配置：保障关键价值链环节资源投入（如研发战略需投入研发资金）
4. 政策制定：制定支持战略的政策与程序（如成本战略需成本控制政策）
5. 绩效与激励：建立与战略目标挂钩的考核、薪酬体系（如创新战略奖励专利）
6. 信息系统：构建支持战略的信息系统（如数字化战略需大数据系统）
7. 价值链优化：提升各环节运作水平（如供应链战略优化物流效率）

### （三）组织结构与战略的关系（钱德勒 “结构跟随战略”）

1. 战略变化→新管理问题→绩效下降→组织结构调整→绩效提升

2. **战略与结构匹配示例**

   | 战略类型     | 组织结构                          | 实例                             |
   | ------------ | --------------------------------- | -------------------------------- |
   | 单一业务战略 | 职能制（研发、生产、营销部门）    | 早期小米（仅手机业务）           |
   | 多元化战略   | 事业部制（按产品 / 区域划分）     | 华为（消费者 BG、运营商 BG）     |
   | 国际化战略   | 区域事业部制 / 全球矩阵制         | 可口可乐（亚太区、欧洲区事业部） |
   | 平台战略     | 网络型结构（核心平台 + 合作伙伴） | 阿里（淘宝平台 + 商家）          |

### （四）战略地图与平衡计分卡（BSC）

1. **战略地图（战略可视化工具）**

   * **四个维度逻辑**：学习与成长（无形资产）→ 内部流程（运营 / 创新）→ 客户（价值主张）→ 财务（股东价值）

   * **各维度核心内容**

     | 维度       | 核心要素                                                 | 实例（苹果）                              |
     | ---------- | -------------------------------------------------------- | ----------------------------------------- |
     | 财务       | 收入增长（新品类）、生产率（成本控制）                   | iPhone+Mac + 服务收入增长，供应链成本控制 |
     | 客户       | 价值主张（价格、质量、品牌）                             | 高端品牌、极致设计、用户体验              |
     | 内部流程   | 运营管理（供应链）、客户管理（体验店）、创新（研发）     | 全球供应链、Apple Store、A 系列芯片研发   |
     | 学习与成长 | 人力资本（员工技能）、信息资本（系统）、组织资本（文化） | 研发人才、iOS 系统、创新文化              |

2. **平衡计分卡（绩效评价工具）**

   * **作用**：将战略转化为可操作的绩效指标，平衡长短期、财务非财务、内外视角。
   * **成功实施条件**：高层承诺、与薪酬挂钩、信息系统支持、持续调整。
   * **实例**：某企业 BSC 指标：财务（ROE、营收增长率）、客户（客户满意度、市场份额）、内部流程（研发周期、良品率）、学习与成长（员工培训小时、专利数量）。

## 十三、典型企业全面战略分析

### 案例一：华为技术有限公司

#### 1. 企业概况与发展历程

* **成立与定位**：1987 年成立，聚焦 ICT 领域（信息与通信技术），愿景 “构建万物互联的智能世界”。
* **发展阶段**：
  * 1987-2000 年（国内拓展期）：从代理交换机到自主研发，积累技术与渠道。
  * 2000-2010 年（国际化初期）：“农村包围城市” 进入新兴市场（东南亚、非洲）。
  * 2010 年至今（全球化领先期）：5G 技术领军者，构建 “硬件 + 软件 + 服务” 生态，业务覆盖 170 + 国家。

#### 2. 核心能力分析（VRIO 标准）

| VRIO 维度                   | 核心能力表现                                                 | 竞争意义                                         |
| --------------------------- | ------------------------------------------------------------ | ------------------------------------------------ |
| 有价值（Value）             | 1. 研发投入：2023 年研发费用超 1900 亿元（占营收 25.1%），5G 专利全球第一；2. 全球供应链：覆盖 100 + 国家，保障芯片 / 设备供应；3. 客户服务：为全球运营商提供端到端解决方案 | 把握 5G 机遇、应对供应链风险、满足客户多样化需求 |
| 稀缺性（Rarity）            | 1. 5G 核心技术：领先竞争对手 1-2 年；2. 全球化组织能力：海外子公司本地化率超 80% | 仅爱立信、诺基亚具备类似能力，形成稀缺优势       |
| 难以模仿性（Inimitability） | 1. 技术积累：30 + 年 ICT 研发沉淀；2. 社会复杂性：与全球运营商 / 政府的长期信任关系；3. 路径依赖：3G→4G→5G 持续迭代 | 模仿成本极高，竞争对手需长期投入                 |
| 可组织利用（Organized）     | 1. 研发体系：“2012 实验室”+“研发军团”；2. 激励机制：股权激励 + 奖金激励；3. 决策机制：轮值董事长制度 | 转化技术 / 供应链资源为竞争优势                  |

#### 3. SWOT 分析与战略匹配

| 维度      | 具体内容                                       | 战略匹配                                                    |
| --------- | ---------------------------------------------- | ----------------------------------------------------------- |
| 优势（S） | 研发领先、全球布局、供应链韧性                 | SO 战略：加大 5G + 工业互联网研发，拓展东南亚 5G 基建       |
| 劣势（W） | 高端芯片依赖、消费者业务短板、海外品牌认知不足 | WO 战略：联合中芯国际突破芯片制程，通过鸿蒙生态提升用户粘性 |
| 机会（O） | 数字经济、新兴市场需求、新基建政策             | ST 战略：多区域供应链规避制裁，新兴市场推高性价比手机       |
| 威胁（T） | 地缘政治制裁、行业竞争、6G 技术风险            | WT 战略：收缩非核心业务，聚焦 ICT 核心领域                  |

#### 4. 战略挑战与建议

* **核心挑战**：芯片自主化、鸿蒙海外渗透、地缘政治风险。
* **建议**：
  1. 技术攻坚：联合国内产业链研发 28nm 成熟制程芯片，长期投入 6G；
  2. 生态开放：降低鸿蒙接入门槛，补贴海外开发者；
  3. 市场多元化：加大拉美 / 中东投入，通过 “技术援助 + 本地化生产” 建立信任。

### 案例二：小米集团

#### 1. 企业概况与发展历程

* **成立与定位**：2010 年成立，初始定位 “为发烧而生”，现愿景 “让全球每个人享受科技美好生活”，聚焦 “手机 ×AIoT” 战略。
* **发展阶段**：
  * 2010-2015 年（手机突围期）：高性价比手机（小米 1 系列）+ 线上渠道，2014 年出货量全球第三。
  * 2015-2020 年（生态链扩张期）：投资 100 + 生态链企业（九号平衡车、米家家电），2018 年香港上市。
  * 2020 年至今（高端化 + 全球化）：小米 11/12 系列冲击高端，欧洲 / 印度市场份额稳居前三。

#### 2. 核心能力分析（VRIO 标准）

| VRIO 维度                   | 核心能力表现                                                 | 竞争意义                                            |
| --------------------------- | ------------------------------------------------------------ | --------------------------------------------------- |
| 有价值（Value）             | 1. IoT 生态：连接设备超 6.55 亿台（不含手机）；2. 成本控制：规模采购 + 代工生产，硬件净利率≤5%；3. 线上渠道：小米商城年 GMV 超 3000 亿元 | 满足 “一站式智能生活” 需求，提供高性价比产品        |
| 稀缺性（Rarity）            | 1. IoT 生态规模：全球第一；2. 米粉文化：社区注册用户超 3 亿，参与产品迭代 | 苹果、华为生态规模不及，粉丝运营难以复制            |
| 难以模仿性（Inimitability） | 1. 生态链管理：“投资 + 赋能” 模式（供应链 / 技术 / 渠道支持）；2. 供应商关系：与高通 / 三星长期合作获优先供货 | 竞争对手需长期积累生态资源，短期难复制              |
| 可组织利用（Organized）     | 1. 生态链架构：生态链部专人对接，提供 “小米方法论”；2. 产品矩阵：“手机 + IoT + 互联网服务” 三层架构 | 转化生态 / 成本优势为盈利（互联网服务毛利率超 40%） |

#### 3. SWOT 分析与战略匹配

| 维度      | 具体内容                                  | 战略匹配                                                |
| --------- | ----------------------------------------- | ------------------------------------------------------- |
| 优势（S） | IoT 生态领先、成本控制、全球化布局        | SO 战略：推出 “全屋智能套餐”，拓展东南亚 AIoT 市场      |
| 劣势（W） | 高端技术短板、线下渠道薄弱、品牌溢价低    | WO 战略：与徕卡深化影像合作，2025 年目标 3 万家小米之家 |
| 机会（O） | AIoT 需求增长、新兴市场潜力、华为高端空缺 | ST 战略：Redmi 应对价格战，小米 Ultra 系列攻高端        |
| 威胁（T） | 行业价格战、鸿蒙生态竞争、政策风险        | WT 战略：收缩低端 IoT 品类，集中资源发展智能汽车        |

#### 4. 战略挑战与建议

* **核心挑战**：高端化受阻、生态协同不足、智能汽车盈利周期长。
* **建议**：
  1. 技术突破：自研影像芯片，与徕卡联合建研发中心；
  2. 生态深化：推出 “小米互联互通协议”，开放给第三方家电；
  3. 汽车聚焦：30-50 万元高端市场，通过 “手机 + 汽车” 协同差异化竞争。
