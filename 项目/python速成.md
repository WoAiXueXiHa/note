# 运算符

就多个 **幂运算 `2**8` $2^8$

`//`整除

`/`带浮点除

`and`就是`&&`

`or`就是`||`

`not`就是`!`

## 容器运算符

`in` 指定容器中查找元素，找到返回`True`

`not in`，没找到`True`



## 身份运算符

`is`两个变量引用自同一个对象返回`True`

`not is`两个变量不是引用自同一个对象返回`True`
