# 对撞双指针

核心思想：

* 选一个基准值，这里我选择数组中间的值作为`key`
* 一个`i`指向索引起始位置，一个`j`指向索引结束位置
* `i`向右找比`key`大的数，找到停下
* `j`向左找比`key`小的数，找到停下
* 交换`i`和`j`对应位置的值，继续下一轮
* 当`i > j`时，`key`左边全是小于`key`的值，`key`右边全是大于`key`的值
* 分治，在子区间中递归执行

代码：

```cpp
void Qsort(vector<int>& nums, int left, int right) {
    if(left >= right) return; // 数组为空或数组只有一个元素，无需排序
    int i = left, j = right;
    int key = nums[left + (right - left) / 2];
    while(i <= j) {
        while(nums[i] < key) i++;	// i找比key大的数，那么循环就设为小，遇到>=就能停下
        while(nums[j] > key) j--;	// j同理
        if(i <= j) swap(nums[i++], nums[j--]);
    }
    // left   j  i  right   第一次循环结束
 	if(i > left) Qsort(nums, left, j);
    if(j < right) Qsort(nums, i, right);
}
```

# 三指针划分

![image-20260421153330094](https://gitee.com/binary-whispers/pic/raw/master///20260421153332412.png)

核心思想不变，还是区间划分

* `i`全量扫描的指针
* `i`从头开始扫描，遇到：
  * `> key` ：`r`要向左把值拉入区间，即是`swap(nums[--r], nums[i])`，**`i`不能`++`！换过去的元素待处理！**
  * `< key`：`l`要向右把值拉入区间，即是`swap(nums[++l], nums[i++])`，这里`i++`，因为左边处理完了
  * `= key`：`i++`就好，扩大`=key`的区间
* 一次循环结束，区间划分成了`[left, l-1]`  `[l, r-1]`  `[r, right]` 只需要处理两边的区间即可

```cpp
void Qsort(vector<int>& nums, int left, int right) {
    if(left >= right) return;
    int i = left, l = left - 1, r = right + 1;
    int key = nums[left + (right - left) / 2];
    while(i < r) {
        if(nums[i] < key) swap(nums[++l], nums[i++]);
        else if(nums[i] == key) i++;
        else swap(nums[--r], nums[i]);
    }
    // left		l		r		right
    Qsort(nums, left, l);
    Qsort(nums, r, right);
} 
```

# 时间复杂度分析

## 良好情况

不管是哪种写法

第一遍的遍历总是`$O(n)$`的全量扫描

接下来的递归深度

$n / 1$	第一层

$n / 2$	第二层

$n / 4$	第三层

...

$n / 2^x$	第x层，子问题规模缩小到了1

那么即是$n / 2^x = 1$ -> $n = 2^x$	-> $x = \log_2n$ -> $O(logn)$

每一层都是$O(n)$，所以总的复杂度是$O(nlogn)$

## 退化情况

如果每次基准值没选好，每次总是一边元素极少，一边元素极多，由于每次划分都极不均匀，比如分成 0 和 n−1，递归树退化成链了

$n+(n−1)+(n−2)+⋯+1=O(n^2)$  这是最坏的复杂度

# 对于选取基准值的优化

这里采用最简单的方式，选取随机数

```cpp
int randNum(vector<int>& nums, int left, int right) {
    int idx = rand() % (right - left + 1) + left;
    return nums[idx];
}
```

对于随机数种子，只需要在`main`函数中种一次就ok了
