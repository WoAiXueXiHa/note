## [283. 移动零 - 力扣（LeetCode）](https://leetcode.cn/problems/move-zeroes/description/?envType=study-plan-v2&envId=top-100-liked)

思路：

> 仿照快排的思路：
>
> `fast`遍历，控制交换权
>
> * 非零交换后`fast++`
> * 0`fast++`
>
> `slow`确保自己及左边区间已经合法

代码：

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = -1, fast = 0;
        int n = nums.size();
        while(fast < n) {
            if(nums[fast] != 0) swap(nums[++slow], nums[fast++]);
            else fast++;
        }
    }
};
```

复杂度：

* 时间：$O(n)$，只需遍历一次区间
* 空间：$O(1)$

## [15. 三数之和 - 力扣（LeetCode）](https://leetcode.cn/problems/3sum/description/?envType=study-plan-v2&envId=top-100-liked)

思路：

> 1. 排序，利用到数组有序性，减少遍历
> 2. 枚举一个数，剩下两个数对撞双指针控制
> 3. 剪枝操作：
>    1. 枚举的要确保不重复
>    2. 答案要确保不重复
>    3. 双指针要确保不重复

代码：

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        int n = nums.size();
        vector<vector<int>> ans;
        if(n < 3) return ans;
        sort(nums.begin(), nums.end());

        for(int i = 0; i < n; i++) {
            // 剪枝1
            if(i > 0 && nums[i] == nums[i - 1]) continue;
            int left = i + 1, right = n - 1;
            while(left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if(sum > 0) right--;            // 大了，找小的值
                else if(sum < 0) left++;        // 小了，找大的值
                else {
                    ans.push_back({nums[i], nums[left], nums[right]});
                    // 保证下一组答案不重复，剪枝2
                    while(left < right && nums[left] == nums[left + 1]) left++;
                    while(left < right && nums[right] == nums[right - 1]) right--;

                    // 寻找下一组解
                    left++;
                    right--;
                }
            }
        }
        return ans;
    }
};
```

复杂度：

* 时间：$O(n^2)$，枚举$O(n)$，双指针遍历一遍$O(n)$，合计$O(n^2)$
* 空间：$O(1)$

## [11. 盛最多水的容器 - 力扣（LeetCode）](https://leetcode.cn/problems/container-with-most-water/description/?envType=study-plan-v2&envId=top-100-liked)

思路：

> **对撞双指针 + 短板效应**
>
> 找到短板之后，就要找比这个短板更大的的短板

代码：

```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int left = 0, right = height.size() - 1;
        int ans = 0;

        while(left < right) {
            // 找到短板，后续不管多大，储水都由短板决定
            // 要找更大的短板，赌我能装更多
            if(height[left] < height[right]) {
                int cur = (right - left) * height[left];
                ans = max(cur, ans);
                left++;
            } else {
                int cur = (right - left) * height[right];
                ans = max(cur, ans);
                right--;
            }
        }
        return ans;
    }
};
```

复杂度：

* 时间：$O(n)$，只需遍历一次区间
* 空间：$O(1)$

## [42. 接雨水 - 力扣（LeetCode）](https://leetcode.cn/problems/trapping-rain-water/description/?envType=study-plan-v2&envId=top-100-liked)

思路：

> **依旧对撞双指针+短板效应**
>
> * 单个位置的水量 = 两侧最高挡板的较小值 - 当前高度
> * 双指针的本质 = 谁的最高挡板更矮，就先结算谁
>
> ![image-20260504120808918](https://gitee.com/binary-whispers/pic/raw/master///20260504120811549.png)

代码：

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int left = 0, right = height.size() - 1;
        int leftMax = 0, rightMax = 0;
        int ans = 0;

        while(left < right) {
            leftMax = max(leftMax, height[left]);
            rightMax = max(rightMax, height[right]);

            if(leftMax < rightMax) {
                ans += leftMax - height[left];
                left++;
            } else {
                ans += rightMax - height[right];
                right--;
            }
        }
        return ans;
    }
};
```

复杂度：

* 时间：$O(n)$，只需遍历一次区间
* 空间：$O(1)$

# 滑动窗口

最长循环外更新结果，最短循环内更新结果

## [3. 无重复字符的最长子串 - 力扣（LeetCode）](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/?envType=study-plan-v2&envId=top-100-liked)

思路：

> 利用一个滑动的窗口维护合法区间：
>
> * 右指针扩充，不断把合法元素往区间内包
> * 左指针确保区间合法，不断右移，吐出去非法元素

代码：

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int n = s.size();
        int hash[128];
        int ans = 0;

        for(int left = 0, right = 0; right < n; right++) {
            hash[s[right]]++;
            while(hash[s[right]] > 1) {
                hash[s[left]]--;
                left++;
            }
            ans = max(ans, right - left + 1);
        }
        return ans;
    }
};
```

复杂度：

* 时间：$O(n)$，只需遍历一次区间
* 空间：$O(1)$，额外的空间是常数级的
